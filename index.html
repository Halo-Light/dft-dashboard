개별 분자의 기저 에너지, ZPE, 진동 모드를 확보하여 Phase 2에서 결합 에너지(ΔEbind) 계산의 기준점 제공. HOMO/LUMO gap으로 전자적 특성 비교.

Phase 2 — Why?

Li+ 주변 용매화 쉘의 열역학적 안정성을 정량화. DEE vs THF의 배위수별 ΔG 차이, TTE 희석제 효과, FSI- CIP 형성 에너지를 측정. 논문의 핵심 데이터.

Phase 3 — Why?

Scientific Discussion

1. Li+ Solvation Thermodynamics: THF > DEE at All Temperatures

233K에서 Li(THF)2의 ΔG = -59.20 kcal/mol vs Li(DEE)2 = -34.73 kcal/mol (ΔΔG = -24.48). 이 차이는 주로 엔탈피 기여(ΔΔH ≈ -27 kcal/mol)에서 기인하며, THF의 고리 구조가 제공하는 최적화된 Li-O 배위 기하학에 의한 것. 온도가 낮아질수록(298→223K) ΔG 차이가 더 벌어짐 — 엔트로피 패널티(TΔS)가 줄어들면서 엔탈피 우세가 강화.

2. TTE Diluent Effect: Selective Destabilization

TTE 도입 시 THF 시스템이 비대칭적으로 불안정화됨. Li(THF)2(TTE)2는 ΔG = +3.65 kcal/mol (233K)로 열역학적 불안정 — 이 조성은 자발적으로 분해. 반면 Li(DEE)1(TTE)2는 ΔG = -9.56 kcal/mol로 여전히 안정. TTE가 THF를 더 효과적으로 밀어냄 → LHCE에서 THF 주변의 TTE 농축이 더 강함.

3. Desolvation Barriers: The Paradox

완료된 5개 스캔의 핵심 결과:

  • LHCE 환경: DEE 탈용매화(17.01) >> THF(6.73) kcal/mol → THF가 더 쉽게 벗겨짐
  • CIP 환경: DEE(9.86) > THF(5.64) kcal/mol → 동일 경향
  • TTE 제거: DEE(6.89) > THF(~0.3) kcal/mol → TTE도 THF계에서 더 쉽게 빠짐

실험적으로 DEE-LHCE가 저온 성능이 우수한데, DFT 탈용매화 장벽은 DEE가 더 높음. 이 역설의 해석: DEE의 우위는 단분자 탈용매화 열역학이 아닌, (i) 더 낮은 점도, (ii) 더 빠른 이온 수송, (iii) SEI 형성 차이에서 비롯될 가능성이 높음. 이는 논문의 주요 논점이 될 수 있음.

4. Entropy Penalty & Temperature Sensitivity

배위수 증가에 따른 TΔS 증가: Li(THF)4에서 TΔS = -33.68 kcal/mol (233K). TTE 혼합 시 엔트로피 패널티 급증 — Li(THF)2(TTE)2에서 TΔS = -34.51. 이는 큰 TTE 분자(MW=252)의 배위 자유도 감소에 기인. 저온에서 엔트로피 효과 감소 → ΔG의 온도 감도: THF계(-11.83 kcal/mol shift, 298→223K)가 DEE계(-8.31)보다 큼.
